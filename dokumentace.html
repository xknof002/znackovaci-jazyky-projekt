Úvod

Jako téma semestrálního projektu k předmětu Značkovací jazyky jsem zpracoval XML s nastavením pro fiktivní počítačovou hru.

Metodika

Využité programy

Pro tvorbu tohoto projektu byl použit program Sublime Text 3 a prohlížeč Google Chrome.

Využité webové stránky

W3Schools, validátory pro XML, XSD a XSLT, HTML validátor, GitHub.

Popis transformace

Transformace do HTML tabulky pomocí cyklu for-each, který projde grafické nastavení všech presetů. HTML tabulka obsahuje porovnání těchto grafických presetů.

Závěr

Při vytváření této semestrální práce jsem se naučil jak vytvořit validní XML schéma a XML soubor s daty k tomuto schématu. Dále jsem se naučil, jak toto schéma a data transformovat do HTML tabulky.
